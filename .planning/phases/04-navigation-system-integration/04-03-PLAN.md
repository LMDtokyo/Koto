PredictiveBackHandler (Activity 1.9+ / Compose, stable):
```kotlin
import androidx.activity.compose.PredictiveBackHandler

// Inside ChatScreen composable:
PredictiveBackHandler(enabled = true) { backEvent ->
    // backEvent is a Flow<BackEventCompat>
    // Collect progress for live preview; on completion (flow completing without exception) = back confirmed
    // Simplest correct implementation:
    backEvent.collect { /* can animate preview using it.progress 0f..1f */ }
    // Flow completes = back gesture committed
    onBack()
}
```
