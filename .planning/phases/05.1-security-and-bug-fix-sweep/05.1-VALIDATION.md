## Wave 0 Requirements

Wave 0 MUST create test infrastructure before coding fixes, otherwise there is no feedback signal. These 8 files/scripts gate subsequent waves:

- [ ] `services/auth/internal/app/service_test.go` — concurrent prekey publish unit test (gates SC-9). Uses in-package fakes per `.planning/codebase/TESTING.md` pattern.
- [ ] `services/bot/internal/http/ssrf_test.go` — URL validation table-driven test (gates SC-3). Cases: `http://169.254.169.254/`, `http://127.0.0.1/`, `http://10.0.0.1/`, `http://localhost/`, `http://postgres:5432/`, `ftp://example.com/`, `https://good-bot.example.com/hook`.
- [ ] `services/gateway/cmd/server/ws_auth_test.go` — WS handshake via `Sec-WebSocket-Protocol` (gates SC-4). Must assert: (a) valid subprotocol → 101 Switching Protocols; (b) missing/invalid subprotocol → 401; (c) response `Sec-WebSocket-Protocol` echoes `koto.auth.v1` only, never the JWT.
- [ ] `services/gateway/internal/middleware/ratelimit_test.go` — unit test for `httprate.LimitByRealIP` behaviour under N concurrent requests.
- [ ] `scripts/test-ratelimit.sh` — integration smoke (gates SC-6). Fires 20 concurrent POST /v1/auth/register, asserts ≥1 response is 429.
- [ ] `scripts/smoke-auth-bundle.sh` — integration smoke (gates SC-2). Curls `GET /v1/auth/prekeys/bundle/abc` without Authorization header, asserts 401 or 404 (never 200).
- [ ] `android/app/src/test/java/run/koto/data/repository/ChatRepositoryTest.kt` — JUnit 4 + MockK (gates SC-1). Verifies `sendMessage()` returns `Result.failure(EncryptionUnavailableException)` when no session exists.
- [ ] `android/app/src/androidTest/java/run/koto/crypto/CryptoManagerInstrumentedTest.kt` — instrumentation test (gates SC-8). Simulates: generate OTKs → consume one via decrypt → persist → restart crypto manager → verify OTK still works for next message.

**Framework install:** None needed. Go `testing` is stdlib. JUnit 4 + MockK (`io.mockk:mockk`) and `kotlinx-coroutines-test` must be added to `app/build.gradle.kts` `testImplementation` during W0 install task.
